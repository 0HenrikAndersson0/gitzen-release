This repository serves as the official distribution channel and issue tracker for the Gitzen application.

## ✨ Key Features

* **⚡ Interactive Graph:** Clear visualization of your commit history, branches, and merges.
* **🎨 Modern Dark Interface:** A sleek, distraction-free environment designed for prolonged use.
* **🛠 Efficient Workflow:** Streamlined staging, unstaging, and discarding of changes.
* **🌿 Branch Control:** Create, merge, and manage branches and tags with ease.
* **🛡 Conflict Management:** Integrated tools to simplify resolving merge conflicts.

## 📥 Download & Installation

Gitzen is currently in **Alpha**. You can download the latest installer for your operating system from the [**Releases Page**](https://github.com/0HenrikAndersson0/gitzen-release/releases).

### Supported Platforms
| Platform | Format |
| :--- | :--- |
| **Linux** | `.deb`, `.AppImage` |
| **Windows** | `.exe` Installer |

## 🐛 Feedback & Issues

While the source code is not public, we highly value your feedback! 

If you encounter a bug or have a feature request, please [**Open an Issue**](https://github.com/0HenrikAndersson0/gitzen-release/issues) in this repository.

**When reporting bugs, please include:**
1.  **Operating System** (e.g., Ubuntu 22.04, Windows 11).
2.  **Gitzen Version** (found in settings or about).
3.  **Steps to Reproduce** the issue.
